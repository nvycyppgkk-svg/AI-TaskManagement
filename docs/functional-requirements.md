## カード

| # | 機能 | 実装状況 |
|---|---|---|
| C-1 | リストにカードを追加できる | 実装済み |
| C-2 | カードを削除できる | 実装済み |
| C-3 | カードにタイトル（必須）を設定できる | 実装済み |
| C-4 | カードに説明文を設定できる | 実装済み |
| C-5 | カードに期限を設定できる | 実装済み |
| C-6 | カードに優先度（高・中・低）を設定できる | 実装済み |
| C-7 | カードにラベル（名前・色）を設定できる | 未実装（カードへのラベル表示のみ。設定・編集は今後の対応） |
| C-8 | カードをドラッグ&ドロップでリスト間の移動・同一リスト内での並び替えができる（手動並び替えモード時） | 実装済み |
| C-10 | リスト単位で優先度順にカードを並び替えられる | 実装済み |
| C-11 | リスト単位で期限順にカードを並び替えられる（期限なしは末尾） | 実装済み |
| C-12 | 並び替えモード（優先度順・期限順）はリストごとに独立して設定できる | 実装済み |
| C-13 | 並び替えモード中はドラッグ&ドロップが無効になる | 実装済み |
| C-9 | カードの詳細をモーダル（ポップアップ）で表示・編集できる | 実装済み |

## エラー時の挙動

| # | 状況 | 挙動 | 実装状況 |
|---|---|---|---|
| E-1 | カードタイトルが未入力で保存しようとした | 保存せず、入力欄にエラーメッセージを表示する | 未実装（今後の対応。現状は保存ボタンが無効化されるのみで、エラーメッセージは表示されない） |
| E-2 | ボード名が未入力で作成しようとした | 作成せず、入力欄にエラーメッセージを表示する | 未実装（今後の対応。ボード作成自体が未実装） |
| E-3 | リスト名が未入力で作成しようとした | 作成せず、入力欄にエラーメッセージを表示する | 未実装（今後の対応。リスト追加自体が未実装） |
| E-4 | ラベル名が未入力で作成しようとした | 作成せず、入力欄にエラーメッセージを表示する | 未実装（今後の対応。ラベル管理自体が未実装） |
| E-5 | ボードを削除しようとした | 確認ダイアログを表示し、確認後に削除する | 未実装（今後の対応。ボード削除自体が未実装） |
| E-6 | カードを削除しようとした | 確認ダイアログを表示し、確認後に削除する | 実装済み |
| E-7 | APIリクエストが失敗した（通信エラー・サーバーエラー） | 画面上部にエラートーストを表示し、操作が保存されなかったことをユーザーに通知する | 実装済み |
